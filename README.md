## ❓ 자주 묻는 질문 (FAQ)

### Q1. 알림이 오지 않아요
**A**: 다음을 확인해주세요:
1. iPhone 설정 → (What2WearNow) → 알림 → 알림 허용 켜기
2. 앱 내 설정 → 알림 → 알림 받기 켜기
3. 알림 시간이 올바르게 설정되어 있는지 확인

### Q2. 위치 정보가 정확하지 않아요
**A**: 
1. iPhone 설정 → 개인정보 보호 및 보안 → 위치 서비스
2. (What2WearNow) → "앱 사용 중" 선택
3. 정확한 위치 허용

### Q3. 날씨 정보가 업데이트되지 않아요
**A**: 
1. 앱 메인 화면에서 새로고침 버튼 (🔄) 클릭
2. 인터넷 연결 상태 확인
3. 앱을 완전히 종료 후 재실행

### Q4. 옷차림 추천이 이상해요
**A**: 
현재 기온을 기준으로 추천해드립니다:
- 23°C 이상: 반팔
- 20°C ~ 22°C: 긴소매
- 12°C ~ 19°C: 후드
- 5°C ~ 11°C: 코트
- 5°C 미만: 패딩

## 🔒 개인정보처리방침

### 수집하는 정보
- **위치 정보**: 현재 위치의 날씨 정보를 제공하기 위해 사용됩니다.
- **알림 설정**: 사용자가 설정한 알림 시간 정보를 기기에 저장합니다.

### 정보 보관
- 모든 정보는 사용자 기기에만 저장되며, 외부 서버로 전송되지 않습니다.
- 앱을 삭제하면 모든 정보가 함께 삭제됩니다.

### 제3자 제공
- 사용자의 개인정보를 제3자에게 제공하지 않습니다.

## 📋 이용약관

### 서비스 이용
본 앱은 날씨 정보 기반 옷차림 추천 서비스를 제공합니다.

### 면책사항
- 날씨 정보는 외부 API(OpenWeatherMap)를 통해 제공되며, 정확성을 보장하지 않습니다.
- 옷차림 추천은 참고용이며, 개인의 체감 온도는 다를 수 있습니다.

### 서비스 변경
개발자는 사전 통지 없이 서비스를 변경하거나 중단할 수 있습니다.
